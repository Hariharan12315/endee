# 🎓 Student Performance Predictor

A Machine Learning project that predicts a student's final performance based on study habits and academic factors. This project demonstrates the use of data analysis, regression models, and a simple web interface for real-time predictions.

---

## 🚀 Features

* 📊 Predict student final score
* 🤖 Machine Learning model using Linear Regression
* 📈 Simple and clean UI using Streamlit
* 📉 Displays performance feedback (Excellent / Average / Needs Improvement)
* 🧠 Easy to extend with advanced ML models

---

## 🛠️ Tech Stack

* Python
* Pandas & NumPy
* Scikit-learn
* Streamlit

## 📊 Dataset

The dataset contains the following features:

* Study Hours
* Attendance (%)
* Previous Score
* Final Score (Target)

You can modify or expand the dataset for better accuracy.

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/student-performance-predictor.git
cd student-performance-predictor
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Train the model

```
python train_model.py
```

### 4. Run the application

```
streamlit run app.py
```

---

## 💻 Usage

* Enter student details such as study hours, attendance, and previous score
* Click on **Predict**
* The model will output the predicted final score along with performance feedback
